###
This is a quick script for collecting hardware and software information from a windows host.
Based on Sydi (https://sydiproject.com)
###

###
Usage

1.	Copy entire directory "INVENTORY" to a USB stick

2.	On each system to be queried, run the INVENTORY.cmd script.

3.	The Inventory results are located in the OUTPUT directory
	a.	INVENTORY.xml (Main System Inventory Report)
	b.	GPRESULT.html (Group Policy Settings)
	c.	SECPOL.inf (Security Policy)
